# CallFlow legal foundation

This folder contains editable legal and informational drafts for the Windows-installable version of CallFlow.

These files are not final legal advice and should be reviewed before publishing a public release or selecting a repository license.

- `TERMS_DRAFT.md`: editable terms and conditions base.
- `PRIVACY_DRAFT.md`: local-data and privacy explanation.
- `LICENSE_OPTIONS.md`: practical comparison before choosing a GitHub license.
- `THIRD_PARTY_NOTICES_DRAFT.md`: third-party notices structure.
- `WINDOWS_INSTALLER_LEGAL_NOTES.md`: notes for the Windows installer and packaging flow.
