Status: **draft — this release has not been published yet.**

## Summary

CallFlow v1.0.0 is the first stable release of the local-first Windows desktop app for call-center operators. It builds on the validated `v0.2.0-beta.1` Official Beta baseline with visual polish, security hardening, documentation finalization, and a complete QA pass.

## Stable scope

CallFlow v1.0.0 covers the following workflows:

- Fast call registration with call types, statuses, outcomes, and custom comments
- CRM copy and supervisor report generation
- Reminder creation, notification, snooze, recurrence, and completion
- Local statistics, timesheets, work/break timers, and estimated earnings
- Multi-timezone clocks with pinned clocks
- Markdown script library with import, export, and PDF handling
- Onboarding wizard with legal acceptance
- Backup export, import, and controlled local data erase
- Dark and light theme support
- Diagnostics and local data health

## Privacy and local-first posture

- All user data is stored locally on the user's computer.
- No cloud sync, backend server, or telemetry in v1.0.0.
- The user is responsible for data they enter, import, copy, export, or store.
- If AI features are implemented, they will not send user data to external services without explicit user consent and updated privacy text.
