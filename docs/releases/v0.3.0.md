# CallFlow v0.3.0

Stable release notes for the first non-beta CallFlow release.

Direct Windows installer:

- https://github.com/GaboEI/callflow/releases/download/v0.3.0/CallFlow.Setup.0.3.0.exe

## Summary

CallFlow v0.3.0 is the new stable baseline for the project. It consolidates the hardening work completed during the beta cycle, keeps the core local workflows intact, and formalizes the release path through GitHub Actions.

## Added

- Public release workflow for tagged Windows builds.
- Stable release notes for `v0.3.0`.
- Renderer test coverage expanded across reminders, knowledge, dashboard, stats, reports, and calculator helpers.

## Changed

- Version bumped from `0.2.0-beta.1` to `0.3.0`.
- Public documentation now presents CallFlow as a stable release line instead of a beta-only snapshot.
- The v1.0 roadmap now uses `v0.3.0` as the baseline for the remaining stable work.

## Fixed

- Durable JSON writes now fsync both the file and its parent directory to reduce loss on crash or power failure.
- Invalid backup JSON now surfaces a clear failure instead of crashing the import path.
- Daily numbering, reminder date handling, calculator evaluation, and renderer helper logic were hardened and covered by tests.

## Release notes

- Windows packaging remains the primary distribution path.
- GitHub Releases is now the canonical place for tagged Windows installer artifacts.
- AI Chat is still a placeholder and remains outside the stable baseline.

## Known limitations

- SmartScreen may still warn if the installer is unsigned.
- AI Chat remains a placeholder.
- Future v1.0 work still includes final polish, legal cleanup, and release candidate QA.
