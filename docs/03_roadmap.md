# Roadmap

## MVP 1: Registro, reportes, CRM, historial

- Electron app shell.
- Onboarding.
- Local settings.
- Call registration.
- CRM copy.
- Hourly report copy.

## MVP 2: Estadísticas y recordatorios

- Daily metric cards.
- Pending reminder counter.
- Reminder creation and completion.
- Overdue reminder grouping.
- Full statistics by primary outcome subtype.
- Full Onboarding and Settings management for success, rejection, and callback presets.

## MVP 3: Biblioteca Scripts Markdown

- Notes CRUD.
- Safe rendered Markdown reader.
- Search.
- Export to `.md` and `.txt`.

## MVP 4: Empaquetado Windows .exe

- Add packaging tooling.
- Add app icon.
- Create unsigned Windows installer or portable build.

## MVP 5: Multiidioma completo y mejoras visuales

- Complete Spanish, English, and Russian translations.
- Improve keyboard flow.
- Add richer empty states.
- Add optional visual themes.
