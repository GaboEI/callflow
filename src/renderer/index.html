Privacidad y datos locales

                    
CallFlow no está diseñado para recopilar datos del usuario ni enviar información a servidores externos. La configuración, llamadas, reportes, scripts y recordatorios se guardan localmente en este equipo.

                    Carpeta de datos: No disponible
